Features

  • 28 Specialized Agents — each with research-backed prompts structured as ROLE / INPUT / OUTPUT / RESPONSIBILITIES / RULES / ANTI-PATTERNS
  • 29 Workflow Presets — from Solo (2 agents, ~$0.04) to Deep Five Minds (27 agents, ~$1-3), covering every orchestration pattern
  • Five Minds Protocol — structured debate with 4 domain experts + Devil's Advocate producing a Gold Solution through adversarial collaboration
  • Human-in-the-Loop Decision Gates — 3 checkpoints between phases with 120s countdown timer, 3 option cards, and auto-fallback
  • Live Pipeline Simulation — agents "talk" to each other with animated speech bubbles, processing spinners, and a full Dialog Timeline log
  • Agent Encyclopedia — encyclopedic knowledge base for every agent and preset: who they are, what they do, what they DON'T do, anti-patterns, fun facts
  • Smart Canvas — drag-and-drop with marquee selection, auto-place, auto-connect, zoom/pan, minimap, and agent suggestion system
  • Dark / Light Theme — full CSS variable system with glassmorphism, floating orbs, starfield animation, and cosmic aesthetic
  • 57 Custom SVG Icons — unique stroke-based icons for all 28 agents and 29 presets with theme-aware dual color palette
  • Final Prompt Generator — one click generates a complete orchestration prompt with colored sections, model badges, and copy-to-clipboard
  • Zero Dependencies — 3,500 lines of vanilla HTML/CSS/JS in a single file. No framework, no bundler, no runtime.

How It Works

                    ┌─────────────┐
                    │ Orchestrator │  ← Central brain (Opus)
                    └──────┬──────┘
                           │
          ┌────────────────┼────────────────┐
          ▼                ▼                ▼
    ┌──────────┐    ┌──────────┐    ┌──────────┐
    │ Research  │    │  Build   │    │    QA    │
    │  Phase    │    │  Phase   │    │  Phase   │
    └────┬─────┘    └────┬─────┘    └────┬─────┘
         │               │               │
    7 Researchers    6 Builders     4 QA Auditors
    in isolation     in parallel    fan-out→aggregate
         │               │               │
         └───────┬───────┘               │
                 ▼                       ▼
          ┌──────────┐           ┌──────────┐
          │Five Minds│           │QA Manager│
          │ Debate   │           │ GO/NO-GO │
          └──────────┘           └──────────┘
  1. Design — Drag agents onto the canvas. Connect them. Choose a preset or build from scratch.
  2. Configure — Set models (Opus/Sonnet/Haiku), edit prompts, define phases.
  3. Simulate — Watch agents communicate in real-time with speech bubbles and phase transitions.
  4. Decide — At HITL gates, choose between 3 options (or let the timer auto-decide in 120s).
  5. Export — Generate a complete orchestration prompt. Copy to clipboard. Paste into Claude Code.
